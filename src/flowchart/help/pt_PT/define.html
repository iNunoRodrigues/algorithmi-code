Definir variáveis

As variáveis são o suporte dos dados em programas computacionais. Representam um espaço na memória do computador que permitem armazer um determinado tipo de dados que varia ao longo da execução do programa.

Variáveis Simples

As variáveis são compostas por:

  • Tipo  : Tipo de dados que são suportados pela variável
    • ​Tipos de dados:
      • ​inteiro - número inteiro - exemplo 1234
      • real - número real - exemplo 3.14
      • caracter - um caracter - exemplo 'a'
      • texto - conjunto de caracteres - exemplo "Ola Mundo"
      • logico - valores Verdadeiro ou Falso
  • Nome : nome pelo qual os dados vão ser acedidos:
    • Regras para a definição de nomes:
      • Só pode conter os caracteres [a..z] , [A..Z], [0..9]
      • Não pode começar por um número [0..9]
      • Não pode ser uma palavra reservada
        • ler, escrever, se, senão, enquanto, etc.
  • Valor : conteúdo da variável. O conteúdo da variável tem de ser compatível com o tipo.

Variáveis compostas

  • Vector(Array uni-dimensional)  : É um conjunto de elementos consecutivos, sendo obrigatoriamente todos do mesmo tipo e é possível aceder individualmente a cada elemento do array.
    • ​Sintaxe geral de um Vector:
      • ​ tipo nome_da_variável [nº de elementos]
        • Exemplo Prático:
          • int v[10] - v é um vector com 10 inteiros.
  • Matriz(array multi-dimensional) : É um vector de vectores, ou seja, é um vector com mais de uma dimensão.
    • ​Sintaxe geral de uma Matriz:
      • ​ tipo nome_da_matriz [dim 1] [dim 2] [dim n]
        • Exemplo Prático:
          • char desenho[3] [3]; - desenho é uma matriz com 3 linhas e cada uma com 3 posições.
  • Registos : conteúdo da variável. O conteúdo da variável tem de ser compatível com o tipo.
